# 🕷️ Layout Spider-Man

Um *layout* responsivo em HTML, CSS (SCSS) e JavaScript inspirado no tema do **🕷️ Homem-Aranha**, aplicado como projeto de frontend para prática e portfólio.

<hr>

🚀 Sobre o Projeto

Esse projeto foi aplicado com objetivo de treinar e demonstrar habilidades em:

- Estruturação de layout responsivo  
- Utilização de **HTML5**, **CSS/SCSS** e **JavaScript**
- Organização de arquivos e pastas em um projeto web

O layout pode ser aberto diretamente no navegador como um **site estático** sem necessidade de servidor.

<hr>

🎓 Créditos

Este projeto foi desenvolvido com base no conteúdo e modelo visual fornecido pelo Codeboots, aplicado como prática para consolidar conhecimentos em desenvolvimento web.
